# Research Papers

## How to Add Your Research Paper

1. **Place your PDF file** in this folder (`public/assets/papers/`)
2. **Name it**: `EGEAT_Paper.pdf`
3. **That's it!** The Research section will automatically display it.

## File Location

```
Portfolio/
  └── public/
      └── assets/
          └── papers/
              └── EGEAT_Paper.pdf  ← Your paper here
```

## Supported Formats

- ✅ PDF (recommended)
- File size: Keep under 10MB for best performance

## Testing

1. Add your PDF to this folder
2. Start dev server: `npm run dev`
3. Navigate to the Research section
4. The PDF should display in the embedded viewer!

---

**Note**: If your PDF has a different name, update the path in `src/components/Research.jsx`
